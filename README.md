# LHC Sensor Predictive Maintenance Project 🚀

## Project Overview
This project simulates sensor data (Temperature, Pressure, Vibration) from high-energy physics experiments to predict equipment failures. As an Industrial Engineering student aiming for CERN, I applied **Machine Learning (Random Forest)** to demonstrate how systematic data analysis can optimize maintenance schedules and minimize downtime.

## 🛠 Technologies Used
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, Seaborn, Matplotlib
* **Algorithm:** Random Forest Classifier (Supervised Learning)

## 📊 Key Insights
* Generated a synthetic dataset representing **1,000 hours** of sensor operation to simulate "Digital Twin" behavior.
* Achieved **100% accuracy** on test data using a multi-sensor threshold logic.
* Analyzed **Feature Importance** to identify that 'Temperature' is the most critical indicator for system reliability.

## 🎯 Why This Matters?
In facilities like CERN, equipment failure leads to costly interruptions. This project serves as a proof-of-concept for using **Predictive Maintenance** to transition from "reactive" repairs to "proactive" system management.
